# downloader.py

an educational overview of how to use yt-dlp to create a command line youtube video/audio downloader. 

It downloads in MP3 for audio and Webm for video

First you need a [venv](https://docs.python.org/3/library/venv.html)

instructions for initalization of the venv are hyperlinked above.

once the venv is initialized and activated, you need to use pip to install yt-dlp

```bash
pip install yt-dlp
```
